## Instalação local
### Requisitos
- Node.js 18.17^

### Passo a passo
1- Para executar o projeto localmente basta executar os seguintes comandos no diretório raiz:

Instalando as dependecias: 
```
$ npm install
```

Iniciando o servidor:
```
$ npm run start
```

2- Após a inicialização do servidor, o app estará rodando em: 
- http://localhost:3000
